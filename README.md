# Guitar Webpage - Strings N Things

A fully responsive e-commerce product showcase page featuring modern web design principles and interactive user experiences.

## Live Demo
[View Live Demo](https://lgxcodes.github.io/Guitar-Webpage/)

## Features

### User Interface
- **Responsive Navigation Bar** with hamburger menu for mobile devices
- **Dropdown Menu System** for product categories (Acoustic Guitars, Electric Guitars, Bass, Amps, Gear)
- **Shopping Cart** with real-time item counter and cart summary
- **Breadcrumb Navigation** for improved user experience

### Product Gallery
- **Interactive Image Gallery** with 6 product views
- **Thumbnail Navigation** with active state indicators
- **Arrow Navigation** for cycling through images
- **Keyboard Support** (left/right arrow keys)
- **Smooth Fade Transitions** between image changes
- **Image Counter** displaying current image position

### Shopping Features
- **Quantity Selector** (1-10 items)
- **Add to Cart Functionality** with instant feedback
- **Cart Management** with itemized view and total calculation
- **Product Specifications** display
- **Dynamic Price Display**

### Technical Features
- **Fully Responsive Design** optimized for all screen sizes
- **CSS Grid & Flexbox Layouts** for modern, flexible design
- **Cross-Browser Compatibility**
- **Mobile-First Approach** with optimized touch interactions
- **Smooth Animations & Transitions**
- **Click-Outside Detection** for menu closing

## Technologies Used

- **HTML** - Semantic markup structure
- **CSS** - Modern styling with Grid, Flexbox, and animations
- **JavaScript** - Interactive functionality and DOM manipulation

## Setup & Installation

1. Clone the repository:
```bash
git clone https://github.com/lgxCodes/Guitar-Webpage.git
```

2. Navigate to the project directory:
```bash
cd Guitar-Webpage
```

3. Open `index.html` in your browser or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js with http-server
npx http-server
```
